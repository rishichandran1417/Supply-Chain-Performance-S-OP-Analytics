# 🔄 Supply Chain Performance & S&OP Analytics

## 📌 Project Overview

This project analyzes supply chain performance using purchase order data across multiple suppliers, SKUs, and regions. The objective is to evaluate supplier reliability, service levels, forecasting effectiveness, and operational efficiency through KPI-driven analytics and Sales & Operations Planning (S&OP) insights.

The analysis was performed using Advanced Excel techniques including Pivot Tables, KPI calculations, lookup functions, and operational performance benchmarking.

---

## 📊 Business Scenario

A company managing multiple suppliers and product categories requires visibility into procurement performance, inventory planning, and service-level compliance.

This project helps answer questions such as:

* Which suppliers consistently miss delivery commitments?
* How accurate are current forecasts?
* Are customer orders being fulfilled in full and on time?
* Which suppliers pose the highest operational risk?
* Where are the biggest planning and service-level gaps?

## 📈 Key Performance Indicators

### OTIF (On-Time In-Full)

Measures the percentage of orders delivered both on time and in full.

**Result:** 32.4%

---

### Fill Rate

Measures the percentage of ordered quantity successfully delivered.

**Result:** 93.6%

---

### Forecast Accuracy

Measures the accuracy of demand planning against actual demand.

**Result:** 90.9%

---

### Lead Time Variance

Measures deviation between expected and actual lead times.

**Result:** +0.51 Days

---

## 🚚 Supplier Performance Analysis

Supplier benchmarking identified performance differences across the supplier network.

| Supplier | OTIF % |
| -------- | ------ |
| SUP001   | 31.2%  |
| SUP002   | 32.0%  |
| SUP003   | 31.1%  |
| SUP004   | 31.6%  |
| SUP005   | 27.4%  |

### Key Finding

⚠️ SUP005 recorded the lowest OTIF performance and represents the highest supplier reliability risk within the network.
